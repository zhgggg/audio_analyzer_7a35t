# 基于 AD7606 的音频分析系统



### 参数概览
| 参数       | 值           |
|------------|--------------|
| 分辨率     | 16 位        |
| 通道数     | 8            |
| 采样速率   | 200 kSPS     |
| 输入范围   | ±5/10 V      |
| 接口类型   | SPI/并行     |

### 特性

- **双极性输入**：无需额外调理电路。
- **高精度**：±5 V 量程下分辨率为 0.153 mV。
- **采样率**：单通道 48 kSPS，适合大多数音频场景。

### 实现

- 使用 AD7606 并口输入，设计转接板适配开发板引脚（嘉立创 EDA）。
- 使用陶晶驰串口屏显示相关信息。
